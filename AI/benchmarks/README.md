# LLM Benchmarks

This project contains tools for benchmarking LLM performance and a web-based UI for visualizing the results.

## Project Structure
- `benchmark.py`: Python script to run benchmarks against Ollama.
- `benchmark_results.json`: Raw results data.
- `ui/`: React-based single-page application for data visualization.

## Quick Start

### Running Benchmarks
1. Install Python dependencies: `pip install -r requirements.txt`
2. Run benchmark: `python benchmark.py`

### Viewing Results
1. Navigate to the UI directory: `cd ui`
2. Install dependencies: `npm install`
3. Start the viewer: `npm run dev`
